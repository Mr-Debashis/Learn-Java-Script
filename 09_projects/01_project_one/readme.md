# Color Switcher App

A minimal and interactive Color Switcher Web App built using HTML, CSS, and Vanilla JavaScript.
This project demonstrates how simple DOM interactions can be used to produce dynamic UI changes with clean, beginner-friendly code.

The app allows users to instantly change the webpage background color by clicking on colored buttons.
